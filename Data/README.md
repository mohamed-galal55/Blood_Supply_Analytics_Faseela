# Data Directory - Faseela Project

## Overview
This directory serves as the central data repository for the Faseela predictive analytics pipeline. It houses both the foundational dimensional models and the dynamically generated machine learning outputs required for operational intelligence and downstream Business Intelligence (BI) integration.

## File Manifest

*   **`Data_Model.xlsx`**
    The core relational data dictionary and primary source file. It contains structured schemas detailing hospital geolocation data, donor demographics and commitment rates, patient clinical requirements, and historical blood stock fulfillment metrics.

*   **`Demand_Forecast_7Days.csv`**
    The time-series predictive output generated by the Prophet forecasting algorithm. This file details the anticipated hospital blood demand over a 7-day horizon, complete with calculated upper and lower statistical confidence intervals.

*   **`AI_Forecast_Results.csv`**
    A comprehensive dataset containing the results of the XGBoost regression and classification models. It includes computed feature engineering metrics such as supply deficit predictions, systemic risk scores, scarcity indices, and urgency probabilities for targeted medical facilities.

*   **`Critical_Cases.csv`**
    A filtered, operationally ready dataset isolating high-risk patient and inventory scenarios. It outputs the specific hospital IDs and blood types algorithmically designated as critical, enabling immediate triage and optimized resource allocation.
